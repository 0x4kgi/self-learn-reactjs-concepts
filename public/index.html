Basic safebooru.donmai.us loader written in ReactJS.

How to use? Enter tags and how many images you want.

So... what are tags? Tags indicate what things, object, or events are in an image

The tags are based on the tagging system in Safebooru/Danbooru.

read more here: Danbooru wiki's tag cheatsheet

Clicking on a thumbnail will redirect you to the actual page where it is hosted
